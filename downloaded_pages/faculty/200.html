Search Tips

Search tabs

There are now three different search tabs, All, Staff and Doctoral Candidates. If someone is both a staff member and doctoral candidate they will display in all three tabs. If you are searching for someone and they are not in your search results, check that you are using the correct tab.  

Partial searches

Partial searches are possible. A search for 'Will', for example, will bring up 'William' First or Family name(s). Results may also include "Will" if it is as part of an email address, username or a profile's keyword or rich content.

Sorting results

At the top of the page is a 'Sort by' drop down menu that allows the search results to be sorted based on 'Last name', 'First name' and 'Relevance'.

Filtering results

Each search result includes the person’s title, department and organisational unit and each one of these can be used to filter results.  By clicking on one or more of the links a filter is added and by removing the original search term a new search list will display with profiles that match the filter, for example if the filter is a title everyone with the same title will display.

Search again 
To perform a new search from a full profile view, use the breadcrumb link "University directory search", which appears just under the banner at the top of the page.
